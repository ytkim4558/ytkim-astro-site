About

클라우드 분석 서비스와 AI Workflow를 기록하는 엔지니어링 노트

AWS Analytics, 데이터 플랫폼, Android/iOS 개발 경험과 AI 코딩 에이전트 기반 작업 흐름을 정리합니다.

안녕하세요. 김용탁입니다.

이 사이트는 클라우드 분석 서비스 운영, 데이터 플랫폼 구축, 그리고 AI 코딩 에이전트를 활용한 개발 워크플로를 기록하는 개인 엔지니어링 노트입니다.

현재는 AWS Analytics 영역에서 Amazon OpenSearch, Kinesis, QuickSight 같은 분석 서비스를 다루고 있습니다. 이전에는 CJ ENM 데이터 플랫폼 구축, OpenShift 기반 마이그레이션, Solr/MongoDB/Redis/RabbitMQ 기반 시스템 운영, Android/iOS 앱 개발 등을 경험했습니다.

이 사이트에 적는 내용

  • AWS Analytics, OpenSearch, 데이터 파이프라인 운영 경험
  • 데이터 플랫폼, 검색 시스템, 분산 시스템 트러블슈팅 기록
  • Claude Code, OpenAI Codex CLI 기반 AI Workflow 실험
  • 개인 도구 제작 과정과 의사결정 기록
  • 오래된 블로그/Jekyll/GitHub Pages 유지보수 기록

읽는 방식

이 사이트는 완성된 튜토리얼 모음이라기보다, 실제 문제를 해결하면서 남긴 엔지니어링 로그에 가깝습니다. 글 안에는 시행착오, 선택지 비교, 실패한 접근, 다음 AI 에이전트에게 넘기는 메모도 같이 남깁니다.

정리된 이력은 Portfolio에서 볼 수 있고, AI 작업 흐름 관련 글은 AI Workflow에 모아두고 있습니다.
